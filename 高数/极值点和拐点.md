**极值点**
- 判别极值的第二充分条件：f' (x 0)=0 & f‘’(x 0)≠0 =>保号性推导
- 判别极值的第三充分条件：n 阶可导，前 n-1 阶导数均为 0，第 n 阶导数不为 0

（1）n 为**偶数**，且第 n 阶导数<0，f (x) 在 x 0 处取得极大值

（2）n 为偶数，且第 n 阶导数>0，f (x) 在 x 0 处取得极小值

- 见到 f (x)·f' (x) 的形式，要想到整体是 f²(x) 求导，从而转换到研究 f²(x) 的性质（5.1）
- f'' (x 0))<0 是 f (x) 在 x 0 处取得极大值的**充分不必要**条件，f'' (x 0)=0 时也可能取得极大值（5.2）

**拐点**

- 极值点只写横坐标，拐点应该写具体坐标
- 判别拐点的第二充分条件：f’' (x 0)=0 & f‘‘’(x 0)≠0 =>保号性推导
- 判别拐点的第三充分条件：n 阶可导，前 2~n-1 阶导数均为 0（**1 阶导可以不为 0**），第 n 阶导数不为 0，**n 为奇数**
