## 🔢 数学一

| 模块 | 分值占比 | 复习重点 |
|:----:|:--------:|:---------|
| 高数 | ~56% | 计算能力是底线，中值定理证明是难点，级数和线面积分是高分题 |
| 线代 | ~22% | 矩阵运算要快准，特征值二次型是核心，向量组秩的概念要清 |
| 概率 | ~22% | 二维随机变量+数字特征是大题常客，参数估计必考 |

### 关键提醒
- ✅ 每天必须动手算，不要只看答案
- ✅ 错题本要定期回顾（第3天、第7天、第30天）
- ✅ 真题要限时做，模拟考场环境
- ❌ 不要沉迷视频课，看懂了≠会做了

### 推荐资料
| 资料 | 用途 |
|:-----|:-----|
| 汤家凤/张宇 基础班 + 辅导讲义 | 一轮打基础 |
| 李永乐线代辅导讲义 | 线代专项 |
| 余丙森概率讲义 | 概率专项 |
| 李林6套卷 + 李林4套卷 | 冲刺模拟 |
| 合工大超越5套卷 | 拔高选做 |
| 历年真题（08-24年） | 核心资料 |

## 🌍 英语一

| 题型 | 分值 | 策略 |
|:----:|:----:|:------|
| 阅读 | 40分 | **得阅读者得天下**。精做+精翻，每篇逐句分析 |
| 作文 | 30分 | 大小作文各准备3-4个模板，灵活套用 |
| 新题型 | 10分 | 掌握七选五/排序/小标题的做题逻辑 |
| 翻译 | 10分 | 抓住主干，调整语序，信达雅 |
| 完形 | 10分 | 性价比低，弱项不必死磕 |

### 关键提醒
- ✅ 单词背诵贯穿始终，从50个/天递增到80-100个/天
- ✅ 阅读真题至少刷2-3遍
- ✅ 作文模板要默写熟练，考场上方能快速输出

### 推荐资料
| 资料 | 用途 |
|:-----|:-----|
| 考研词汇闪过 / 红宝书 | 单词贯穿全程 |
| 唐迟阅读方法论 | 阅读技巧 |
| 颉斌斌阅读 | 阅读技巧（可选） |
| 张剑黄皮书真题 | 真题精做 |
| 石雷鹏作文 | 作文模板 |
| 真题00-24年 | 核心资料（近10年重点） |

## 🧪 408 计算机学科专业基础

| 科目 | 分值 | 特点 |
|:----:|:----:|:------|
| DS | 45分 | 算法题是拉分项，代码题要手写熟练 |
| CO | 45分 | 硬件概念多，CPU流水线是大题重点 |
| OS | 35分 | PV操作大题年年考，内存管理要理解透彻 |
| CN | 25分 | 分值少但知识点杂，TCP/IP协议栈要清晰 |

### 关键提醒
- ✅ 选择题考得很细，教材要读透
- ✅ 大题注重综合应用，多练习真题
- ✅ 数据结构代码题要手写，不要只在脑子里过
- ✅ 概念辨析题（如各种调度算法区别）要整理对比表格

### 推荐资料
| 资料 | 用途 |
|:-----|:-----|
| 王道408四科辅导书 | 核心教材，覆盖全面 |
| 严蔚敏《数据结构》教材 | DS深入理解 |
| 唐朔飞《计算机组成原理》教材 | CO深入理解 |
| 408历年真题（09-24年） | 核心资料 |
| 王道模拟题 | 选择题专项练习 |

| 时政 | 16分 | ⏰ 11月集中 |

### 关键提醒
- ✅ **选择题是生命线**（单选16分+多选34分=50分），决定政治分数上限
