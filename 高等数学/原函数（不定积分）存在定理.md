#不定积分 
## 原函数（不定积分）存在定理

### 1. 核心定理（充分非必要）

若函数 $f(x)$ 在区间 $I$ 上**连续**，则 $f(x)$ 在 $I$ 上**存在原函数**。

此时，变上限积分
$$F(x)=\int_a^x f(t)\,dt \quad (a,x\in I)$$
就是 $f(x)$ 的一个原函数，满足：
$$F'(x)=f(x)$$

---

### 2. 定理的局限性

连续 **只是充分条件**，**不是必要条件**。

| 情况        | 是否有原函数          | 典型例子                                      |
| --------- | --------------- | ----------------------------------------- |
| $f$ 连续    | ✅ 必有            | $e^{x^2},\ \sin x$                        |
| 跳跃间断点     | ❌ 无（[[达布定理（导函数介值定理）]]禁止） | $\text{sgn}(x)$                           |
| 可去间断点     | ❌ 无             | $f(x)=\frac{\sin x}{x}\ (x\ne0),\ f(0)=1$ |
| 振荡间断点（无界） | ❌ 无             | $\frac1x$ 在 $x=0$ 附近                      |
| 振荡间断点（有界） | ✅ 可以有           | $2x\sin\frac1x-\cos\frac1x$，补充 $f(0)=0$   |

> **核心结论：** 有原函数的函数未必连续，但**一定具有介值性（达布性质）**，因此**不能有跳跃间断点**。

---

### 3. 与定积分存在的对比（易混淆点）

|          | 原函数存在             | 定积分（黎曼可积）存在       |
| -------- | ----------------- | ----------------- |
| **充分条件** | 连续                | 连续 / 有界且仅有限个间断点   |
| **要求强弱** | 强（导数必须满足介值性）      | 弱（允许跳跃间断）         |
| **关系**   | 有原函数 **不一定** 黎曼可积 | 黎曼可积 **不一定** 有原函数 |

---

### 4. 记忆口诀

> **连续必有原，变上限来算；**
> **间断分两半，跳跃全完蛋，**
> **可去也不行，振荡看情况。**
