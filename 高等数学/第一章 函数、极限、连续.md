# 第二讲 数列极限
- **数学归纳法**（~~2.1~~）
1. 第一数学归纳法
（1）验证n=1时命题成立
（2）假设n=k时命题成立
（3）证明n=k+1时命题成立

2. 第二数学归纳法
（1）验证n=1和n=2时命题成立
（2）假设n<k时命题成立
（3）验证n=k时命题成立

- **海涅定理(p130)**：证明函数极限不存在/计算数列极限
- 求数列极限通常要做一两步逆运算，并且**联想经典形式**（[[../错题/高数_01_30讲_2.11|高数_一_30讲_2.11]]）
- 利用**单调有界准则**求极限（~~2.12~~）
- 单调有界准则+数学归纳法（[[../错题/高数_01_30讲_2.14|高数_01_30讲_2]]）（注：2.1+2.12的综合）
- [[压缩映射原理]]（**p134**)求数列极限（[[../错题/高数_01_30讲_2.15|高数_01_30讲_2]]）
- 数列极限定义的ε可以替换成不依赖n的任意小的正数，不能与n有关（[[../错题/高数_01_30讲_2.17|高数_01_30讲_2]]）

## 洛必达
 [【26 考研数学 | 经典错误系列 #2 】一个视频扫清洛必达雷区！](https://www.bilibili.com/video/BV19yZ2YrEXa/?share_source=copy_web&vd_source=ea4f51f7e64e6fd33c534d494301cb4e)

1. **后验性**：洛必达所求极限不存在，只能说明洛必达法则失效，不能说明极限不存在
2. **广义洛必达**：∞/∞型只需要判断分母是∞，就可以洛必达
3. **二阶可导≠二阶导函数极限存在**=>不可用洛必达=>用导数定义
4. **数列极限无法洛必达**，需要通过海涅定理先转换成函数极限问题

## 等价无穷小
[[[https://www.bilibili.com/video/BV1sZZHYdEzF/?share_source=copy_web&vd_source=ea4f51f7e64e6fd33c534d494301cb4e](https://www.bilibili.com/video/BV1sZZHYdEzF/?share_source=copy_web&vd_source=ea4f51f7e64e6fd33c534d494301cb4e|【【26考研数学 | 经典错误系列#1】什么时候能用等价无穷小？】 ]]

1. 趋于0，**不能取到0**
2. 加减中，替换后**分子的阶数≥分母阶数**
3. 部分求极限，需要该部分**极限=A≠0**
4. 加减可以**对数化**后变成乘除，然后再用等价无穷小

## 反三角函数
### 反三角函数
![[Pasted image 20260612043944.png]]
- 三角函数**一拱面积**为2
- $\sin (\arccos x)=\sqrt{1-x^2} ,\quad\cos (\arcsin x) = \sqrt{1 - x^2}$
- $\arcsin x+\arccos x=\frac{\pi}{2},\quad(−1\leq x \leq 1)$
- $\arctan x+\operatorname{arccot} x=\frac{\pi}{2},\quad(-\infty\leq x\leq+\infty)$ 
### 双曲函数
1. 函数式：$\operatorname{arsinh} x = \ln\left(x + \sqrt{x^2 + 1}\right), \quad x \in (-\infty, +\infty)$
2. 性质：奇函数，x的等价无穷小
3. 积分公式$\int \frac{1}{\sqrt{1 + x^2}} dx = \operatorname{arsinh} x + C = \ln\left (x + \sqrt{x^2 + 1}\right) + C$
