以下是 Markdown 语法的大全，涵盖常用和扩展语法，适合笔记、文档、博客等场景。

## 一、标题
```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 二、段落与换行
- 直接写文字即为段落。
- 两个空格 + 回车 = 换行（有的编辑器用两个回车换段）。

## 三、强调

| 样式 | 语法 | 示例 |
|------|------|------|
| 斜体 | `*斜体*` 或 `_斜体_` | *斜体* |
| 粗体 | `**粗体**` 或 `__粗体__` | **粗体** |
| 粗斜体 | `***粗斜体***` | ***粗斜体*** |
| 删除线 | `~~删除线~~` | ~~删除线~~ |
| 下划线 | 不支持标准语法，可用 HTML `<u>` | <u>下划线</u> |

## 四、列表

### 无序列表（- 或 * 或 +）
```markdown
- 苹果
- 香蕉
  - 子项（缩进2-4空格）
* 橙子
+ 葡萄
```

### 有序列表（数字 + 点）
```markdown
1. 第一项
2. 第二项
   3. 子项1
   4. 子项2
5. 第三项
```

### 任务列表
```markdown
- [x] 已完成任务
- [ ] 未完成任务
```

## 五、链接

```markdown
[文字](https://example.com "标题（可选）")

<https://example.com>  自动链接

[引用][id]  然后在某处定义：[id]: https://example.com
```

## 六、图片

```markdown
![替代文本](image.jpg "图片标题")
![本地图片](./images/pic.png)
```

## 七、引用

```markdown
> 这是一段引用
> 可以多行
>> 嵌套引用
```

## 八、代码

### 行内代码
`` `code` ``

### 代码块
````markdown
```python
print("Hello")
```
````

常用语言标识：`python`、`javascript`、`html`、`css`、`bash`、`json` 等。

## 九、分隔线

```markdown
---  或 *** 或 ___
```

## 十、表格

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 苹果   | 红色     | 5元    |
| 香蕉   | 黄色     | 3元    |
```

`:---` 左对齐，`:---:` 居中，`---:` 右对齐。

## 十一、转义字符
用反斜杠转义特殊字符：
```markdown
\*不是斜体\*
\# 不是标题
```

可转义：`\`、`` ` ``、`*`、`_`、`{}`、`[]`、`()`、`#`、`+`、`-`、`.`、`!`、`|`、`>`、`~`。

## 十二、HTML 支持
大部分 Markdown 解析器允许内嵌 HTML：
```html
<kbd>Ctrl</kbd> + <kbd>C</kbd>
<br> 换行
<div style="color: red;">红字</div>
```

## 十三、扩展语法（因平台而异）

| 功能 | 语法 |
|------|------|
| 脚注 | `[^1]`，然后 `[^1]: 说明文字` |
| 定义列表 | `术语`<br>`: 定义`（某些平台） |
| 数学公式 | `$E=mc^2$`（需要 LaTeX 支持） |
| 图表/流程图 | Mermaid、PlantUML 等（如 GitHub、Typora） |
| 高亮 | `==高亮文字==`（部分扩展） |
| 目录 | `[TOC]` 或 `<!-- TOC -->`（取决于编辑器） |

## 十四、常见平台注意点

- **GitHub**：支持任务列表、脚注、代码块语言、Mermaid
- **Typora**：支持几乎所有扩展，包括数学、高亮、上下标
- **Obsidian**：支持 Wiki 链接 `[[页面名]]`
- **知乎、简书**：基础语法多数支持，扩展功能有限
